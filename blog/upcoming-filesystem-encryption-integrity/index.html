Upcoming encryption and integrity filesystem realm support in Elytron

By Ashpan Raskar | December 14, 2021

Two features that we have been working on for Elytron security is the ability to encrypt filesystem security realms with a SecretKey, and the ability to verify filesystem integrity with a KeyPair. This feature is planned to be released in WildFly 27.
